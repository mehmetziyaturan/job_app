# Job App
job applications were tracked in the job tracking application, made with redux toolkit, 
applications are listed, we can delete them, has detailed filtering, different sorting, insertion process. 

# Libraries
- sass
- @reduxjs/toolkit
- react-redux
- react-router-dom
- react-toastify
- json-server
- axios
